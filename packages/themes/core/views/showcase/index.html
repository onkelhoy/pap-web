Core Theme Overview

Welcome to the core theme for Papit components. The focus here is simplicity, harnessing the power of CSS's new relative colors. This approach allows for more intuitive design adjustments and fewer variables.

Components like buttons can now inherit base colors and dynamically adjust them for states like hover and active, using CSS's relative color functions.

Each Color Includes:

  • The main color, defined by its variable (e.g., --pap-primary)
  • Text color for use with the main color (e.g., --pap-primary-text)
  • Contrast color for text or elements on top of the main color (e.g., --pap-primary-contrast)

Say goodbye to dozens of specific-purpose variables, and embrace streamlined, flexible color management!
